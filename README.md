# vscode-snippets-fe README

个人总结前端开发常用代码片段（通常是比较大的片段），包括但不限于style、js、vue、html、react等

## Features
业务中常用的大块代码片段不用从旧项目复制粘贴，直接一键输入。输入前缀，即可显示候选代码片段。
